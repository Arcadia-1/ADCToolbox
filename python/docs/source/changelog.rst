Version 0.3.0
-------------

Current release with comprehensive ADC analysis capabilities.

Features
~~~~~~~~

* 21 ready-to-run examples across 6 categories
* Analog output analysis (9 diagnostic tools)
* Digital output analysis (6 tools)
* Dual MATLAB and Python implementations with numerical parity
* CI/CD enabled with 100% test coverage
* Full documentation

Key Capabilities
~~~~~~~~~~~~~~~~

* **Analog Analysis**: INL/DNL, harmonic decomposition, error PDF/spectrum/autocorrelation
* **Digital Analysis**: Bit weight calibration, overflow detection, bit activity
* **Spectrum Tools**: FFT metrics, SFDR, ENOB, polar plots
* **Signal Generation**: Coherent frequency calculation, test signals
* **Oversampling**: Delta-Sigma modulator analysis
